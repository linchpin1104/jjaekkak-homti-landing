# 째깍홈티 빠른 시작 가이드

5분 안에 째깍홈티를 배포하고 테스트해보세요!

## 🚀 빠른 배포 (5분)

### 1️⃣ Google Sheets 준비 (1분)

1. https://sheets.google.com 접속
2. 빈 스프레드시트 생성
3. 시트 2개 만들기: "부모신청", "치료사신청"
4. URL에서 스프레드시트 ID 복사

```
https://docs.google.com/spreadsheets/d/[여기가_ID]/edit
```

### 2️⃣ Apps Script 설정 (2분)

1. 스프레드시트에서 **확장 프로그램 > Apps Script**
2. `google-apps-script/Code.gs` 파일 전체 복사/붙여넣기
3. 5번째 줄 수정:
   ```javascript
   const SPREADSHEET_ID = '여기에_복사한_ID_붙여넣기';
   ```
4. **배포 > 새 배포 > 웹 앱**
   - 액세스: **모든 사용자**
5. 웹 앱 URL 복사

### 3️⃣ 프론트엔드 연결 (30초)

1. `assets/js/form-handler.js` 파일 열기
2. 2번째 줄 수정:
   ```javascript
   const GOOGLE_SCRIPT_URL = '여기에_웹앱_URL_붙여넣기';
   ```
3. 저장

### 4️⃣ Netlify 배포 (1분 30초)

1. https://netlify.com 접속 및 가입
2. **Sites > Add new site > Deploy manually**
3. `landing-pages` 폴더 드래그 앤 드롭
4. 완료! 🎉

---

## 🧪 테스트 (3분)

### 1️⃣ 브라우저에서 확인

배포된 URL로 접속:

```
https://your-site.netlify.app
```

### 2️⃣ 부모 신청 테스트

1. **부모님** 카드 클릭
2. **무료 상담 신청하기** 클릭
3. 테스트 정보 입력:
   - 보호자명: 테스트
   - 연락처: 010-1234-5678
   - 아이 나이: 만 5세
   - 진단명: 테스트
   - (나머지 필수 항목 입력)
4. **무료 상담 신청하기** 버튼 클릭
5. "신청이 완료되었습니다!" 확인

### 3️⃣ 데이터 확인

1. Google Sheets 열기
2. "부모신청" 시트에서 방금 입력한 데이터 확인

### 4️⃣ 치료사 신청 테스트

같은 방식으로 치료사 지원서 제출 테스트

## 🆘 문제 발생?

### 폼이 제출되지 않아요
- F12 눌러서 콘솔 확인
- `form-handler.js`의 URL 재확인

### Sheets에 저장 안 돼요
- Apps Script의 스프레드시트 ID 재확인
- 시트 이름 확인: "부모신청", "치료사신청" (띄어쓰기 없음)

### 더 자세한 도움이 필요해요
- `DEPLOYMENT_GUIDE.md` 참고
- `README.md` 문제 해결 섹션 확인
