1. **사진/이미지 추가**
   - 째깍악어 앱 화면 레퍼런스 추가
   - 아이들과 선생님이 잘 지내는 느낌의 영역 추가
   - 교사 프로필 화면 이미지 추가

2. **프로모션 강조 (매우 중요!)**

3. **로고 제작 및 반영**
   - 로고 디자인 필요

### 📝 부모 페이지 수정사항

#### 프로세스 수정
**기존 (5단계):**
1. 온라인 신청서 작성 (3분)
2. 운영팀의 1:1 상담 (전화/카톡) ❌ 삭제
3. 우리 아이에게 맞는 치료사 추천 + 교사프로필 전달 ⭐
4. 프로필 확인 후 최종 결정 + 리뷰와 후기 볼 수 있음 ⭐
5. 첫 수업 일정 조율 및 시작

**변경 후 (4단계):**
1. 온라인 신청서 작성
2. 우리 아이에게 맞는 홈티 선생님 추천 (교사 프로필 전달)
3. 프로필 확인 후 최종 결정 (리뷰와 후기 확인 가능)
4. 첫 수업 일정 조율 및 시작

#### 버튼 & 헤더 수정
- **"무료 상담 신청하기"** → **"서비스 오픈 알림 받기"**
- **신청서 헤더**: "무료 상담 신청" 문구 및 설명 전체 삭제
- **베타테스트 오픈 예정: 3월 22일** 안내 추가

#### 폼 필드 수정
- **추가**: 아이 이름 필드
- **필수 → 선택 변경**: 
  - 선호하는 연락 방법 (*)
  - 대부분의 필드를 선택으로 변경 (매칭 품질 향상 목적)
- **용어 변경**: "치료" → "홈티"

### 👨‍⚕️ 치료사 페이지 수정사항

#### 수입 계산 섹션 강조
**기존:**
- 단순 수입 예시

**변경:**
- **비교 강조**: 
  ```
  다른 플랫폼 (아이홈티/가치톡)
  월 매출: 1,280,000원
  수수료 (평균 7.5%): -96,000원
  실수령: 1,184,000원
  
  째깍홈티 (3월말 사전등록)
  월 매출: 1,280,000원
  수수료: 0원 (첫 3개월) ⭐
  실수령: 1,280,000원
  
  → 3개월간 288,000원 세이브!
  ```
- **원천징수 별도 표기**
- **3개월 이후 수수료 정책 명시**

#### 버튼 수정
- **"치료사 지원하기"** → **"서비스 오픈 알림 받기"**

#### 용어 변경
- "치료" → "홈티"
